Gnostic is a personal knowledge managment system which archives the things you read online, created during MadHacks 2025.



### **gnostic-extension**

The tool used for configuring how often/how granular the archival is done. By default, any webpages that are viewed for 10 seconds are archived.


### **gnostic-website**

A browser for the personal knowledge database


### **gnosticdb**

The python package containing the server which can be run with `python3 -m gnosticdb`
